Welcome to Giraffe's documentation!
============================================
Giraffe |version|
============================================

Giraffe is a graph feature tool that provides the following features:

* Efficient graph feature computation including PageRank, Katz, various centralities
* Community detection methods such as Louvain
* Facilities to create, filter, update time-aware graphs
