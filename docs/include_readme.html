Giraffe - A Tool for Graph Feature Generation

Giraffe is a graph feature tool that provides the following features:

  • Efficient graph feature computation including PageRank, Katz, various centralities

  • Community detection methods such as Louvain

  • Facilities to create, filter, update time-aware graphs

This README aims to provide a general introduction to Giraffe.

Installation

Follow the steps below to install Giraffe.

  1. It is recommended to create a virtual environment using virtualenv so that problems of dependencies and versions could be better addressed.

  2. Clone the repository from the remote

    git clone https://github.com/sparkle-apt/giraffe.git

  3. Change to the Giraffe directory

    cd giraffe

  4. Install dependencies required by Giraffe if they do not exist locally

    pip install -r requirements.txt

  5. Install Giraffe

    python setup.py install --use

  6. Testing

    Try importing Giraffe package in your local Python environment:

    >>> import giraffe

    If you see the following error appear,

    Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ModuleNotFoundError: No module named 'giraffe'

    it is likely that the installation is not correct.

    Otherwise, congratulations! You have successfully installed and configured Giraffe.

The installation duration depends on network condition, computer configuration, as well as whether dependencies are fulfilled or not. If the dependencies have been already fulfilled, the installation should take only several minutes.

Voilà!
